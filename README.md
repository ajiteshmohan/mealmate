# 🍽️ MealMate

A modern food delivery web application built with HTML, CSS, and JavaScript.

---

## 📁 Project Structure

```
MealMate/
├── index.html        # Main HTML file (app entry point)
├── styles.css        # Stylesheet (UI, layout, animations)
├── script.js         # JavaScript logic (cart, offers, OTP sign-in)
└── logo.png          # MealMate brand logo
```

---

## ✨ Features

- 🛒 **Cart Panel** — Add, remove, and manage food items
- 🎟️ **Offers Modal** — Browse and apply available deals
- 📱 **Phone OTP Sign-In** — Secure login via one-time password

---

## 🚀 How to Run Locally

1. Clone the repository:
   ```bash
   git clone https://github.com/YOUR_USERNAME/mealmate.git
   cd mealmate
   ```

2. Open `index.html` in your browser — no server needed!

## 🔄 How to Update Your GitHub Repository

Follow these steps every time you make changes to your code:

### Step 1 — Open Terminal
Navigate to your project folder:
```bash
cd path/to/your/mealmate
```

### Step 2 — Check What Changed
```bash
git status
```
This shows which files were modified, added, or deleted.

### Step 3 — Stage Your Changes
To stage all changed files:
```bash
git add .
```
Or stage a specific file only:
```bash
git add index.html
git add styles.css
git add script.js
```

### Step 4 — Commit with a Message
```bash
git commit -m "Your message describing what you changed"
```
**Example commit messages:**
- `"Add offers modal UI"`
- `"Fix cart item count bug"`
- `"Update OTP sign-in flow"`

### Step 5 — Push to GitHub
```bash
git push origin main
```
> If your branch is named `master` instead of `main`, use:
> ```bash
> git push origin master
> ```
